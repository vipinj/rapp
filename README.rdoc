== README

   Dummy Application for deployment and load testing.
   100 x 100 Identity Matrix multiplication.

* Ruby version

  2.1.1p76

* System dependencies

  nodejs for a js runtime, dev packages for 
  sqlite-dev(el), 
  curl-dev(el)

* Configuration

  Please see bootme.sh

* Deployment instructions

  Requires port 80 to be open  on the aws security configuration.
  <10rps is enough for saturating an AWS micro-ami compute capacity.
